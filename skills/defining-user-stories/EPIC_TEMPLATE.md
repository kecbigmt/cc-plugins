# Epic Template

Copy this template to create a new epic document.

```markdown
## Epic: [Epic Name]

### Vision
[One paragraph describing what this epic aims to achieve and why it matters]

### User Value
[What value does this epic deliver to users when complete?]

### Stories

#### Phase 1: [Phase Name]
1. **[Story Name]** - [Brief description]
   - As a [role], I want [capability], so that [benefit]

2. **[Story Name]** - [Brief description]
   - As a [role], I want [capability], so that [benefit]

#### Phase 2: [Phase Name]
3. **[Story Name]** - [Brief description]
   - As a [role], I want [capability], so that [benefit]

### Out of Scope
- [Features explicitly not included in this epic]
- [Future enhancements that are separate efforts]

### Dependencies
- [External dependencies or prerequisites]
- [Other epics or stories that must complete first]

### Risks & Considerations
- [Technical risks]
- [Design decisions that need resolution]
- [Areas of uncertainty]

---

### Progress

| Story | Status | Story Log |
|-------|--------|-----------|
| [Story 1] | Not Started | - |
| [Story 2] | Not Started | - |

### Notes
[Running notes about the epic as it progresses]
```
